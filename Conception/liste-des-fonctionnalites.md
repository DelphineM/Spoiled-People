# Liste des fonctionnalités attendues

## Gestion de compte

- Création
- Modification
- Suppression

Un compte =
- identifiant / mot de passe
- adresse(s) postale(s)
- paramètres
- visibilié par défaut
- adresse par défaut
- notifications
- notification par défaut
- notifications à propos de SpoiledPeople

## Gestion de liste

- Ajout
- Modification
- Suppression
  
Une liste =
- nom
- visibilité (privée / partagée / publique)
- date de l'événement
- adresse liée
- notification
- un ou des contacts attachés
  
## Gestion d'item

- Ajout
	- via galerie mobile
	- via appareil photo mobile
	- via image téléchargée
	- via URL
	- via code-barre
	- via bookmarklet
	- manuellement
- Modification
- Suppression

Un item =
- titre et/ou image
- description
- prix
- quantité souhaitée
- statut
  
## Gestion de contact

- Ajout d'une nouvelle relation
- Suppression d'une relation
